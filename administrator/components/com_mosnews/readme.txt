MosNews is design to display today's news by default.  It also provide option for user to select such as:
. Dropdown menu to select the existing news base on year (yy), month (yyyy-mm) and date (yyyy-mm-dd)
. Menu bar with selection of < Month - < Week - < Date -  Date > -  Week > -  Month > ranging from
  previous month, week, date to next date, week and month
. User can have total control of how it look by using CSS in your template
